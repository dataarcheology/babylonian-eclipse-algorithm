# babylonian-eclipse-algorithm
Algoritma Türü: Babylonian System B (Linear Zigzag Function).Temel Denklem: Saros Rezonansı ($223 S_\lambda = 242 D_\lambda$).Sütun F (Column F): Bu kod, Babil tabletlerindeki "Lunar Latitude" sütununu (Sütun F) simüle eder. Babilliler sinüs dalgası bilmezdi; bunun yerine Ay'ın yörüngesini keskin iniş çıkışları olan "Zigzag" fonksiyonlarla modellerdi.
